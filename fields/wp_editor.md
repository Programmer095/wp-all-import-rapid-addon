# wp_editor

## Description

Provide a WP Editor field.

## Usage

### **Example**

```php
$my_addon->add_field( 'my_wp_editor_field', 'My WP Editor Field', 'wp_editor',  null, 'Tooltip', false, 'Default Text' );
```
